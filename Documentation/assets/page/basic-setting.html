      Basic Setting

      Administrative Site Control Hub

      Site Preferences:
      • Site Name :
        Administrators can set and modify the site name, establishing a distinct and recognizable identity for the crowdfunding platform.
      • Time Region :
        Set the application's time zone, ensuring accurate and synchronized timing across the entire system.
      • Item Showing Unit :
        This will decide that how many of items will be shown in one table page.

      System Preferences:
      • Email Alert :
        Enable or disable email alerts for various system notifications, providing flexibility in user communication.
      • SMS Alert :
        Control the activation of SMS alerts for users, facilitating efficient communication.
      • Enforce SSL :
        Enhance platform security by enforcing SSL connections for encrypted data transmission.

      Logo & Favicon Preferences:
      • Light and Dark Logo :
        Customize the platform's appearance with distinct light and dark versions of the site logo.
      • Favicon Icon :
        Set the favicon icon, contributing to brand recognition and user experience.
